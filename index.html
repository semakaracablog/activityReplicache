The next web is

Replicache is a Javascript framework for building high-performance,
offiline-capable, collaborative web apps.

Benefits

Realtime collaboration

Users see each others' changes live, as they
happen. No refreshes, no locking.

Instant UI

Say goodbye to roundtrips and progress bars. All reads and writes are "optimistic" by default.

Offline support

Applications can go offline for hours or days
and sync up smoothly when they come back online.

BYOB(ackend)

Replicache is client-side technology and works with many backend stacks.

It just works

Conflicts,tabs,schema migrations - Replicache handles the hard parts. You'll find it's
even more productive than traditional web frameworks
